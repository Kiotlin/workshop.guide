# 做项目时常用的工具

## Emojipedia

<AppCard picOrgin="https://lnks.ga/xiK62Z" appName="Emojipedia" website="https://emojipedia.org/">
    表情图标百科是一个表情图标参考网站，记载了Unicode标准中各个表情图标的编码、含义、演变等信息。 网站同时也是
    Unicode联盟的一员，提供了Unicode网站上的部分彩色表情图标。
</AppCard>

<Callout>
    表情包集装箱，直接 ``Ctrl + C```即可。
</Callout>
